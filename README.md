# arbeitsblatt
Stylesheet for TeX

## Uses
- Class: exam
- babel (naustrian)
- inputenc (utf8)
- color

## Options:
- answers, noanswers: show/hide answers
- A, B, C, D: Shows group A/B/C/D on header
- M, PH: Shows Mathematics/Physics on footer
- 1, 2, 3, 4, 5, 6, 7, 8: Shows grade 1-8
- sans: Sets font to sans
